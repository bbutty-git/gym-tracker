☁️ One-time cloud setup

~5 minutes. This gives you a free online database (Supabase) that stores your workouts and syncs them across devices. No Google, no server, nothing that can be blocked. You only do this once.
  1. Go to supabase.com → Start your project → sign up (free) → New project. Pick a name, a region near you, and a database password (save it somewhere).
  2. Wait ~2 min for it to finish setting up. Then open SQL Editor (left sidebar) → New query → paste the snippet below → Run.
  3. Open Authentication → URL Configuration. Set Site URL to https://bbutty-git.github.io/gym-tracker/ and add that same address under Redirect URLs. (This is what lets the email sign-in link bring you back to the app.)
  4. Open Project Settings → API. Copy the Project URL and the anon public key.
  5. Paste both into the two fields in Settings, then type your email and tap “Email me a sign-in link.” Click the link in your inbox → you're synced. 🎉

    
The anon key is designed to be public and is protected by the security rule in that SQL — only a signed-in account can read its own rows. It's safe to paste into the app.
